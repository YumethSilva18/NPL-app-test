║                          REQUEST FLOW                                    ║
╚══════════════════════════════════════════════════════════════════════════╝

Frontend → POST /api/predict
    ↓
Security Layer (Helmet, CORS, Rate Limit)
    ↓
Request ID Middleware (Generate UUID)
    ↓
Logger Middleware (Log Request)
    ↓
[JWT Auth Guard] (Optional - Currently Disabled)
    ↓
Validation Pipe (Validate DTO - 16 features)
    ↓
PredictionController.predict()
    ↓
PredictionService.createPrediction()
    ├─→ Preprocessing (Feature Engineering)
    ├─→ ModelService.predict()
    │   ├─→ HTTP POST to ML Service
    │   ├─→ [If Failed] → Rule-Based Fallback
    │   └─→ Return Prediction
    ├─→ Postprocessing
    │   ├─→ Determine Risk Level (Low/Medium/High)
    │   ├─→ Generate Decision Hint (Approve/Review/Escalate/Decline)
    │   ├─→ Calculate Score Breakdown (5 components)
    │   └─→ Generate Recommendation Text
    └─→ PredictionRepository.create()
        └─→ Prisma → PostgreSQL (Save)
    ↓
Transform Interceptor (Wrap Response)
    ↓
Audit Interceptor (Log to audit_logs)
    ↓
Logger Middleware (Log Response)
    ↓
Response → Frontend (JSON with snake_case)
